# TYPO

**🔗 Live Site:** [https://typo.kushnaik364.workers.dev/](https://typo.kushnaik364.workers.dev/)

A kinetic typing speed game built with React, Tailwind CSS, and Framer Motion. Test your typing skills across three difficulty levels with real-time WPM and accuracy tracking.

## Features

- **3 Difficulty Levels** with distinct typing requirements:
  - **Easy** — All lowercase, no punctuation. Long flowing sentences.
  - **Medium** — Uppercase letters + simple punctuation (`, . '`).
  - **Hard** — Uppercase letters + complex punctuation (`! ? ; : — " '`).
- **150-word sequences** generated by concatenating random quotes per difficulty
- **60-second timeout** — game auto-ends and shows results
- **Real-time WPM & accuracy** tracking during gameplay
- **localStorage high scores** — top 6 per difficulty, displayed as a 2×3 grid
- **Latest stats marquee** — shows your most recent game metrics at the top
- **Completion modal** — animated stats summary with Play Again / Back to Home
- **Keyboard shortcuts** — Enter to start, Escape to quit

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Design

Kinetic Typography aesthetic — brutalist, high-contrast dark theme with acid yellow accent (#DFE104), aggressive uppercase typography using Space Grotesk, infinite marquee, sharp corners (0px radius), and Framer Motion animations.

## How It Works

1. **Landing** — Pick a difficulty or press Enter to start with Medium difficulty
2. **Type** — 150 words appear on screen; type each word and press Space to advance
3. **Track** — WPM, accuracy, and elapsed time update in real-time
4. **Finish** — Complete all words or hit the "esc" key to quit; review stats in the modal
5. **Repeat** — Play again or dismiss to home; high scores persist via localStorage
